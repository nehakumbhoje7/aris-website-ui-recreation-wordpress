# Aris-Website-UI-Recreation-Wordpress

## Project Type
Practice / Learning Project

## Disclaimer
This project is a UI recreation of the publicly available ARIS website.
It was created strictly for educational and learning purposes.
All design and content rights belong to the original owners.

## Tools Used
- WordPress
- OceanWP Theme
- Elementor Page Builder
- WPForms Plugin

## Work Done
- Installed WordPress locally using XAMPP
- Configured OceanWP theme
- Recreated website pages using Elementor
- Managed responsive layout
- Created contact form using WPForms
- Managed menus and media

## Learning Outcomes
- Understanding WordPress structure
- Hands-on experience with Elementor
- Theme customization
- Plugin integration
- Responsive web design

## How to Run Locally
1. Install XAMPP
2. Copy project folder to htdocs
3. Import database using phpMyAdmin
4. Update wp-config.php
5. Open http://localhost/myweb
